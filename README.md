# Run

1. `npm install -g pnpm`
1. `pnpm install`
1. `pnpm run serve`

# Create JS export of SASS variables

Since there is no great way to use SASS variables directly in JS, we compile the SASS variables to a JS object.

1. `pnpm build:tokens`
